总结

基本上这样个人的博客就算搭建完毕了，以后要撰写博客只需要进行以下几个步骤

  1. 在E:\blog\blog_posts目录下新建md文件
  2. 开始撰写文章
  3. 执行命令 hexo g
  4. git commit and git push
